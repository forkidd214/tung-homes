# Tung Homes

A demo website for a local builder.

## Overview

- Next.js 15 + React 19
- Mobile-first with Tailwind CSS
- TypeScript + ESLint + Prettier
- GitHub Actions CI/CD
- Docker containerization
- Playwright E2E testing
- Radix UI components

## Development

```bash
pnpm install
pnpm dev
pnpm test:e2e
pnpm build
```

## Structure

- `/src` - Source code
- `/public` - Static files
- `/e2e` - Tests
- `/components` - UI components
